🧠 Key Takeaways

  • AD Recycle Bin: Deleted users can be restored and abused if permissions allow.
  • Kerberoasting with custom SPN: Adding a fake SPN to a writable user makes them kerberoastable.
  • VSIX extensions as RCE vector: VS Code extensions can execute arbitrary code when installed.
  • BadSuccessor dMSA: Abusing DMSA to get RC4 hash of a service account.
  • Volatility memory forensics: Extracting credentials from VM snapshots.
