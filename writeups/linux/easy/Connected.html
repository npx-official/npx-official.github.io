📊 Key Takeaways

  • Unauthenticated SQLi does not require a login to cause full compromise
  • EXTRACTVALUE truncates at 32 chars — use SUBSTRING to extract long values
  • SQLi write == authentication bypass — UPDATE ampusers SET password_sha1
  • incron + world-writable trigger == privesc — ha_trigger is world-writable
  • PHP module hijacking — root-level script loads PHP from user-writable path
