🔒 Security Lessons

  • Never use MailHog in production — It exposes all emails publicly without authentication.
  • Password reset tokens should expire — Tokens should be single-use and time-limited.
  • Don't store secrets in environment variables — Use a secrets manager.
  • Credential reuse is dangerous — SMTP password reused for host SSH access.
  • Never run services as root — Gogs running as root enabled filesystem-level attacks.
  • Symlink validation is critical — Gogs failed to resolve symlinks safely before writing content.
