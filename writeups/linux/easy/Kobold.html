📊 Key Takeaways

  • Never trust user input in API endpoints. MCP API accepted arbitrary commands without validation.
  • Docker group membership = root. Never add regular users to the docker group.
  • Use rootless containers. Prevents this escalation path.
  • Network segmentation. The MCP API should not be exposed without authentication.
