🚀 HTB CCTV - Complete Walkthrough

Machine: CCTV Difficulty: 🟢 Easy OS: 🐧 Linux

📋 Machine Information

CCTV is an Easy difficulty Linux machine featuring a web application with a command injection vulnerability.

🔍 Reconnaissance

Port scan reveals HTTP service on port 80. Web enumeration discovers a CCTV monitoring dashboard.

🎯 Exploitation

The web application is vulnerable to command injection via the ip parameter.

👑 Privilege Escalation

After gaining initial access, the user can escalate privileges using a cron job running as root.

🏁 Flag

User flag and root flag are located in the standard directories.
